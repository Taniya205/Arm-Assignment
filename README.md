# Taniya Enterprise Website

.Taniya Enterprise Website is a business website built using React. It showcases the company’s services including crane hiring, crane repairing, lubricants, and spare parts, with dedicated pages for Home, About, Services, and Contact. The website is fully responsive and easy to use on both mobile and desktop

## Tech Used
- React
- React Router
- React Bootstrap
- CSS

## How to Run the Project
1. Install dependencies
npm install

2. Start the development server
npm start


Then open in your browser:

http://localhost:3000
